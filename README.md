# üè° Rent Rover ‚Äì Find Your Home

A modern rental marketplace app built for Bangladesh, developed using **Kotlin** and **Firebase**. Rent Rover allows tenants to find suitable homes and landlords to post rental listings easily. The app includes secure authentication, real-time chat, smart filtering, and a smooth user experience from search to communication.

---

## App Features

### **1. Splash Screen**
A clean welcome screen displaying the Rent Rover logo while the app initializes.

### **2. Sign In Screen**
Existing users can log in using email and password. Includes ‚ÄúForgot Password?‚Äù support and Google sign-in.

### **3. Reset Password Screen**
Users enter their email to receive a password reset link. A minimal and focused design for quick account recovery.

### **4. Sign Up Screen**
Allows new users to create an account by providing name, email, and password. Includes  a ‚ÄúRemember me‚Äù option.

##  User Home & Search

### **5. Home Page **
Shows a scrollable list of rental properties with photos, rent amount, location, and basic specs.

### **6. Search Bar  Filter Screen*
Users can quickly search properties by keywords or use the filter panel to refine results. The filter allows selecting location (Division ‚Üí District ‚Üí Upazilla), property type (Family Apartment, Boys' Mess, Girls' Mess), available facilities (Gas, Wifi, Parking, CCTV), and setting a budget range.

## Posting a Rental Circular

### **9. Post Circular **
The Post Circular form lets landlords upload property photos, set the location, choose the property type, add specifications, select facilities, write a short description, and provide rent and contact details. After completion, the listing can be published with a single tap.
